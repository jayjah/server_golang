### GoLang Server Example Project with GIN
    Simple GoLang Server to learn Golang for myself

#### Basic commands:

##### Build
```
    go build main.go
```

##### Run
```
    go run main.go
```

##### Run with Hot-Reload
```
    air
```

##### Create/Recreate db models in `./models`
```
    sqlboiler psql -c sqlboiler.toml
```
